The following API calls are available from this back-end.

  1. Segment Satellite Image: /segment/URL Replace URL with the image URL you want to run segmentation on. Returns URL of the Segmented image.
  2. Get Results: /results/INDEX Replace index with the index of the chosen roof-top. Returns JSON comprising results based on the area of chosen roof.
  3. Others: To be added based on the project requirements.
